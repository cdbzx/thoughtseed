# 灵芽（ThoughtSeed）Review Checklist

## 产品一致性
- 是否仍然围绕“记录、补全、复习、转化”闭环？
- 是否引入了不必要的复杂功能？

## 体验一致性
- 快速记录是否足够短？
- 详情页是否过重？
- 复习流程是否清晰？

## 技术一致性
- 是否保持 Web-first + PWA 策略？
- 是否保持 AI provider 可切换？
- 是否避免过早复杂化？

## 代码质量
- API 是否清晰简单？
- 页面职责是否清楚？
- 命名是否统一？
- 是否有明显重复代码？

## 发布准备
- README 是否可读？
- 环境变量是否齐全？
- 本地启动是否清楚？
