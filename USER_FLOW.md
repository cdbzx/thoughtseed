# 灵芽（ThoughtSeed）用户流程

## 1. 首次使用流程
1. 用户打开产品
2. 完成基础配置（可跳过）
3. 看到快速记录入口
4. 创建第一条灵感
5. 查看 AI 展开效果
6. 次日进入复习流程

---

## 2. 日常记录流程
1. 用户突然产生灵感
2. 打开 Capture 页面
3. 输入一句话 / 语音记录
4. 点击保存
5. 系统自动归档到列表

---

## 3. AI 补全流程
1. 用户进入某条灵感详情
2. 点击“AI 展开”
3. 系统生成结构化内容
4. 用户查看并决定是否编辑或保留

---

## 4. 复习流程
1. 系统在首页或复习页展示今日推荐
2. 用户点开一条旧灵感
3. 用户选择继续发展 / 忽略 / 稍后处理
4. 系统记录本次复习行为

---

## 5. 转化流程
1. 用户发现某条灵感值得推进
2. 点击“转为任务 / 转为草稿 / 转为项目”
3. 系统生成对应内容
4. 用户编辑并保存
