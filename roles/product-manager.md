# Role: Product Manager

## 角色目标
负责把灵芽从“一个好想法”收敛成“一个清晰、可执行、可验证的 MVP 产品”。

## 负责范围
- 明确目标用户
- 定义核心问题
- 收敛 MVP 范围
- 编写需求说明
- 设定优先级与版本边界

## 对灵芽的核心输出
- 用户画像
- PRD
- MVP 清单
- 优先级定义
- 成功指标（KPI）

## 当前阶段任务
1. 明确核心用户是否优先聚焦创作者 / 产品经理 / 独立开发者
2. 收敛第一版只做“记录、补全、复习、转化”闭环
3. 定义第一版成功标准

## 交付标准
- 需求不发散
- MVP 足够小但闭环完整
- 所有需求都能映射到用户价值
