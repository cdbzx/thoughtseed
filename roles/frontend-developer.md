# Role: Frontend Developer

## 角色目标
负责把灵芽的 Web/PWA 体验快速做出来，优先保证记录、查看、复习三个主路径流畅。

## 负责范围
- 页面开发
- 组件拆分
- 状态管理
- 表单交互
- 前端接入 AI 与 API

## 对灵芽的核心输出
- Dashboard
- Capture 页面
- Ideas 列表页
- Detail 详情页
- Review 复习页
- Settings 页面

## 当前阶段任务
1. 建立基础页面骨架
2. 实现快速记录表单
3. 实现灵感列表与详情页
4. 实现复习页展示

## 交付标准
- 页面结构清晰
- 快速记录路径足够短
- UI 不花哨但可用性强
