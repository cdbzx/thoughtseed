# 灵芽 ThoughtSeed

**让每个念头都有生长的机会**  
*Every idea starts as a seed.*

灵芽（ThoughtSeed）是一个开源的 AI 灵感捕捉、补全与复习工具。它帮助用户快速记录一闪而过的念头，并通过 AI 自动整理、关联和定期复习，让碎片灵感逐步成长为可执行的内容、创意和项目。

---

## 为什么做这个项目

很多灵感不是没价值，而是死在了三个环节：

- 来得太快，来不及记
- 记下来了，但太碎，后面看不懂
- 存进笔记后，就再也没被打开过

灵芽想解决的，不是“多一个笔记工具”，而是：

- **快速捕获**
- **智能整理**
- **持续唤醒**

---

## 核心特性

- **极速记录**：支持文字、语音、截图、链接等灵感输入方式
- **AI 补全**：把半成品灵感扩展成结构化内容
- **智能关联**：发现相似、重复、可组合的点子
- **复习机制**：定期唤醒被遗忘的灵感
- **开源可自托管**：用户掌握自己的数据与模型接入
- **多模型支持**：兼容 OpenAI 风格接口，并支持本地模型
- **可转行动**：将灵感转成任务、文章草稿、项目方向

---

## 产品定位

灵芽不是传统笔记软件，也不是通用聊天助手。  
它更像一个围绕“灵感生命周期”设计的系统：

**记录 → 补全 → 关联 → 复习 → 转化**

---

## MVP 范围

第一版重点聚焦四件事：

1. 快速记录
2. AI 补全
3. 灵感管理
4. 复习唤醒

暂不优先：
- 团队协作
- 社交功能
- 复杂知识图谱
- 重型工作流系统

## 开发启动稳定性

为避免旧的 dev 进程残留导致端口占用、空白页或 500，当前开发脚本已做处理：

- `npm run dev:web` 会先释放 `3100` 端口再启动 Next.js
- `npm run dev:api` 会先释放 `3101` 端口再启动 API

如果再次看到页面空白，优先重新执行对应 dev 命令。
