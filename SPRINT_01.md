# 灵芽（ThoughtSeed）Sprint 01

## Sprint 目标
在第一周内做出一个可演示的最小闭环：
- 能记录灵感
- 能查看灵感
- 能对单条灵感做 AI 展开
- 能看到基础复习入口

---

## Sprint 范围
### 必做
- 项目初始化
- 基础页面骨架
- Idea 数据表
- 新增 / 列表 / 详情 API
- 快速记录页面
- 灵感列表页面
- 灵感详情页面
- AI 展开接口
- 基础复习页静态版

### 不做
- 完整权限体系
- 高级搜索
- 语义相似推荐
- 导出功能
- 多端同步

---

## Sprint 验收标准
- 用户可新增一条灵感
- 用户可在列表看到历史灵感
- 用户可查看某条灵感详情
- 用户可点击 AI 展开并拿到结构化内容
- 页面可本地跑通

---

## 建议节奏
### Day 1
- 初始化仓库与项目结构
- 建库与基础表结构
- 搭首页 / Capture / List 骨架

### Day 2
- 完成新增和列表 API
- 接通快速记录页

### Day 3
- 完成详情页
- 完成 AI 展开接口

### Day 4
- 接通详情页 AI 展开展示
- 做复习页静态结构

### Day 5
- 联调
- 修 bug
- 更新 README 与启动说明
